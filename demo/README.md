# FlowTrack Demo Barcode Assets

These assets are aligned with the products loaded by Settings > Demo data > Sync demo data.

Use `qa-barcode-sheet.svg` when you want one printable sheet. Use the individual PNG files in `barcodes/` when you want to show one barcode at a time on another screen.
